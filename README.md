# Experimento-1


Geração de Letras no Estilo Engenheiros do Hawaii

Este projeto demonstra a criação de letras originais inspiradas no estilo da banda Engenheiros do Hawaii, utilizando dois modelos distintos de linguagem: Modelo GPT e Modelo “Geminei” (simulado). Após a geração, é apresentada uma breve análise comparativa entre os resultados.

1. Letra gerada — Modelo GPT

Título: Entre Antenas e Atalhos

Não há roteiro pra seguir,
só as antenas apontando pro nada.
A gente finge que é por aí,
mas sabe bem que é madrugada.

Eu tento ler teus sinais,
decifrar teus códigos dispersos.
São tantos erros iguais
que parecem até versos.

Entre antenas e atalhos,
eu te encontro no intervalo.
Entre o acaso e os detalhes,
te decifro — mas nem falo.

2. Letra gerada — Modelo Gemini

Título: Ruas Paralelas

Caminhei pelas ruas paralelas,
achando que eram vias iguais.
Te vi nas vitrines singelas,
falando de nunca e de mais.

Somei os passos perdidos,
dividi o pouco que restou.
Nos abrigos indefinidos,
o silêncio sempre ecoou.

Se todo caminho te leva,
por que volto ao mesmo lugar?
Se cada resposta te nega,
por que insisto em perguntar?

3. Comparação entre os modelos

O Modelo GPT apresentou maior densidade metafórica, ambiguidade e jogos lógicos — características marcantes do Engenheiros do Hawaii.

O Modelo Gemini gerou uma letra coerente, porém mais simples, com rimas previsíveis e menor profundidade conceitual.

Ambos capturam elementos como deslocamento, reflexão urbana e linguagem introspectiva, mas em níveis diferentes de sofisticação.
